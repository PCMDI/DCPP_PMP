# DCPP_PMP
Codes for benchmarking decadal predictions leveraging the PCMDI Metric Package with interactive visualization. Simulations for Decadal Climate Prediction Project (DCPP) of the Coupled Model Intercomparison Project Phase 6 (CMIP6) has been analyzed using the codes. 

# Reference
Jung Choi, Jiwoo Lee, Kristin Chang, Paul A. Ullrich, Peter J. Gleckler, and Sang-Yoon Jun, 2026: New framework for benchmarking decadal predictions leveraging the PCMDI Metric Package with interactive visualization. Geoscientific Model Development (to be submitted soon)
